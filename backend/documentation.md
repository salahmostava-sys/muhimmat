# API Documentation

- `src/controllers/`

- `src/middlewares/`

## Current Status

- Supabase backend logic is preserved under `backend/supabase/`.
- `server.js` is present at backend root as the backend entry point placeholder.
- Feature-specific route/controller/model files for Employees, Orders, Salary, Attendance, Platforms, and Roles are not yet present in this repository and need to be added when backend application code is introduced.
